# Improvements Made

This repo was prepared as a GitHub/Vercel-ready static app.

## Repo readiness

- Added a direct `index.html` entry point at the repository root.
- Added `README.md` with usage, deployment, limitations, and disclaimers.
- Added `LICENSE` using All Rights Reserved language.
- Added `.gitignore` for macOS junk, local build artifacts, environment files, and logs.
- Added `.nojekyll` so GitHub Pages will serve files without Jekyll processing.
- Added `vercel.json` with static-friendly clean URLs and basic security headers.

## App readiness

- Single-file static HTML/CSS/JS app.
- No npm install required.
- No build command required.
- Progress persists in browser `localStorage`.
- Includes practice questions, flashcards, quick labs, PBQ-style simulations, confidence tracking, and JSON progress export.

## Content/compliance readiness

- Added clear disclaimer that the project is not affiliated with WGU, CompTIA, AWS, LPI, or PeopleCert.
- Added clear note that the project does not include exam dumps, real exam questions, or protected exam content.
- Added known gaps before serious use.
- Added CompTIA Cloud+ as a visible gap/coverage module.
